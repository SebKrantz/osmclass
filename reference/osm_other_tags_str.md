# Extract Remaining 'other_tags' as Comma-Separated Strings

Parses the 'other_tags' column from an imported OSM PBF file and
reformats it as a comma-separated `key:"value"` string per feature,
optionally dropping a set of tag keys. The intended use is to preserve
OSM tags not already captured by
[`osm_classify()`](https://sebkrantz.github.io/osmclass/reference/osm_classify.md):
pass the classification's tag keys via `exclude` so they are not
duplicated in `main_tag`, `main_tag_value`, and `alt_tags_values`.

## Usage

``` r
osm_other_tags_str(x, exclude = character(0), split = "\",\"|\"=>\"", ...)
```

## Arguments

- x:

  character. The 'other_tags' column of an imported osm.pbf file.

- exclude:

  character. Tag keys to drop from the output. Default `character(0)`.

- split:

  character. Pattern passed to
  [`strsplit`](https://rdrr.io/r/base/strsplit.html) to split up `x`.

- ...:

  further arguments to
  [`strsplit`](https://rdrr.io/r/base/strsplit.html).

## Value

a character vector of the same length as `x`, with each element a
comma-separated string of `key:"value"` pairs, or `NA_character_` if no
tags remain after exclusion (or the input was missing/empty).

## See also

[`osm_other_tags_list`](https://sebkrantz.github.io/osmclass/reference/osm_other_tags_list.md),
[`osm_classify`](https://sebkrantz.github.io/osmclass/reference/osm_classify.md),
[osmclass-package](https://sebkrantz.github.io/osmclass/reference/osmclass-package.md)
